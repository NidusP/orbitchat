# Orbitchat Backend

## Overview

Orbitchat backend built with Bun and Hono.

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build
pnpm build

# Lint
pnpm lint

# Type check
pnpm type-check
```

## Structure

- `src/index.ts` - Application entry point
- `src/routes/` - API routes (future)
- `src/middleware/` - Middleware functions (future)
- `src/services/` - Business logic (future)
- `src/lib/` - Utilities and helpers (future)
- `src/types/` - Local type definitions (future)
