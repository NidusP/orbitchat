# Orbitchat Web Frontend

## Overview

Orbitchat web frontend built with Next.js 14 and React 18.

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build
pnpm build

# Start production server
pnpm start

# Lint
pnpm lint

# Type check
pnpm type-check
```

## Structure

- `src/app/` - App Router pages and layouts
- `src/components/` - React components
- `src/hooks/` - Custom hooks
- `src/lib/` - Utilities (API calls, validators, etc.)
- `src/types/` - Local type definitions
- `src/styles/` - Global styles
